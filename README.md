  Telegram Quiz Bot
  Описание

  Бот представляет собой Telegram-квиз, который задаёт пользователю вопросы и оценивает ответы.
Результаты сохраняются с привязкой к ID пользователя, чтобы отслеживать статистику прохождений.


1. Вставьте ваш токен тг-бота в config.py.
2. Напишите свои вопросы в questions.py.
3. Установите зависимости из requirements.txt.
4. Запускайте и пользуйтесь)). (py main.py)
